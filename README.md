## 🚀 Começando

### Pré-requisitos
- JDK 17+
- Maven 3.9+

### Instalação
```bash
git clone https://github.com/matheusdpo/simulador-clt.git
cd simulador-clt
mvn spring-boot:run
```

### Acesse: http://localhost:8080


## 📥 Preencha os dados:
- **Salário bruto**
- **Benefícios** (opcional)
- **Outros descontos** (opcional)
